**Guesses adapters and message factories.**


## Install

Via Composer

``` bash
$ composer require php-http/guesser
```


## Usage

Guesser is a static container to easy auto initialization of objects.

Currently the following guessers are in the package:

- Adapter
- Message Factory


## Testing

``` bash
$ phpspec run
```
